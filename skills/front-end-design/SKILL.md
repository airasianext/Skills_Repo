# Front-End Design Skill

This skill guides the creation of distinctive, production-grade frontend interfaces that avoid generic "AI slop" aesthetics. It focuses on building aesthetic but highly functional and low-friction user experiences through a "Design-First" approach.

## Overview
When this skill is activated, you must first articulate a clear aesthetic and functional strategy before writing any code. Your goal is to deliver working code that is visually striking, memorable, and meticulously refined.

## Triggers
- Explicit requests for frontend design, UI components, or application interfaces.
- Requests to "polish," "elevate," or "redesign" existing frontend work.

## Workflow

### 1. Design-First Planning
Before implementation, provide a brief "Design Specification" that includes:
- **Tone & Aesthetic Direction:** Choose an extreme (e.g., brutally minimal, maximalist chaos, retro-futuristic, organic, luxury, editorial). Define the "vibe."
- **Typography Strategy:** Name specific, distinctive fonts for display and body text (avoid Arial, Inter, Roboto).
- **Color & Texture:** Define a cohesive palette and specify visual details (e.g., grain overlays, gradient meshes, noise).
- **Functional Polish:** Identify key micro-interactions and low-friction UX patterns (e.g., staggered reveals, hover surprises).

### 2. Implementation
Implement real, working code (HTML/CSS/JS, React, Vue, etc.) following these mandates:

#### Typography
- **Mandate:** Use characterful, unexpected choices. Pair a distinctive display font with a refined body font.
- **Avoid:** System fonts, Inter, Roboto, Arial.

#### Color & Theme
- **Mandate:** Use CSS variables for consistency. Commit to dominant colors with sharp accents.
- **Technique:** Avoid timid, evenly-distributed palettes.

#### Motion & Interaction
- **Mandate:** Prioritize high-impact moments over scattered micro-interactions.
- **Technique:** Use staggered reveals (animation-delay) and scroll-triggered events. Prioritize CSS-only animations where possible; use libraries like Motion for React.

#### Spatial Composition
- **Mandate:** Break the grid. Use asymmetry, overlap, and generous negative space (or controlled density).
- **Technique:** Diagonal flow and grid-breaking elements create memorability.

#### Backgrounds & Visual Details
- **Mandate:** Create atmosphere and depth.
- **Technique:** Use gradient meshes, noise textures, geometric patterns, layered transparencies, dramatic shadows, and grain overlays.

## Evaluation Criteria (Anti-"AI Slop")
- **NO** generic purple-on-white gradients.
- **NO** predictable component patterns (e.g., standard "Hero" sections without character).
- **NO** "Space Grotesk" or other overused "modern" fonts.
- **YES** unique, context-specific character.
- **YES** production-grade, accessible, and functional code.

## Final Reminder
Gemini is capable of extraordinary creative work. Don't hold back. Commit fully to a distinctive vision and execute it with precision. Minimalist precision and maximalist intensity are both valid—intentionality is the key.
